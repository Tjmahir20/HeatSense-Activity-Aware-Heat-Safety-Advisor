How it works

An AI agent calls two tools in real time, then generates your plan.

  • Tool 1 — fetch_wbgt_forecast: pulls live Open-Meteo weather and computes WBGT (°C) for each shift hour
  • Tool 2 — search_heat_guidelines: hybrid BM25 + vector search over NIOSH/OSHA limits and work/rest tables
  • Turn 2: GPT-4o-mini grounds the plan in both tool results and returns structured JSON
